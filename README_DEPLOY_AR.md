# AloGo Global Landing Page

هذا موقع هبوط احترافي جاهز للنشر لـ AloGo Smart Travel.

## الملفات

- `index.html` الصفحة الرئيسية.
- `styles.css` التصميم والحركات والتجاوب.
- `script.js` تبديل اللغات والحركات الذكية.
- `assets/` شعار AloGo والصور المجهزة من الشعار الذي أرسلته.

## طريقة الرفع السريعة

### cPanel / استضافة عادية
1. افتح File Manager.
2. ادخل إلى `public_html` أو مجلد الدومين `alogo.online`.
3. ارفع كل الملفات الموجودة داخل هذا المجلد.
4. تأكد أن `index.html` موجود في الجذر.

### GitHub Pages
1. أنشئ مستودعًا جديدًا مثل `alogo-website`.
2. ارفع ملفات الموقع كما هي.
3. من Settings > Pages اختر الفرع `main` والمجلد `/root`.
4. اربط الدومين `www.alogo.online` من إعدادات Pages.

## تعديل الروابط

داخل `index.html` ابحث عن:

- `https://www.alogo.online`
- `mailto:hello@alogo.online`

واستبدلها بروابط التطبيق أو البريد الحقيقي عند الإطلاق.

## اللغات المدعومة

العربية، الإنجليزية، الفرنسية، الألمانية، الإسبانية، التركية، الرومانية.
